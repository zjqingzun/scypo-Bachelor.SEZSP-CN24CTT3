# SOCKET PROGRAMMING


## Introduction
__Architecture:__ Client - Server <br>
__Protocol:__ RSTP (TCP) - RTP (UDP)


## Usage
### Clone Repository
```
git clone https://github.com/zjqingzun/scypo-Bachelor.SEZSP-CN24CTT3.git
```

### Navigate to Directory
```
cd scypo-Bachelor.SEZSP-CN24CTT3
```

### Check Environment
```
python --version
pip --version
pip install -r requirements.txt
```

### Run Project
```
cd src
```
### 1. Server
```
cd server
python Server.py 1234
```
### 2. Client
```
cd client
python ClientLauncher.py 127.0.0.1 1234 444 movie.Mjpeg
```
